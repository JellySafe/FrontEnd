# Claude Code Rules

- 프로젝트 루트에서 Claude Code를 실행한다.
- 여러 패키지나 파일을 수정하는 작업은 먼저 계획을 제시한다.
- 변경 예정 파일을 먼저 알린 뒤 작업한다.
- Figma 작업은 `AGENTS.md`의 Figma workflow를 따른다.
- 디자인 시스템을 먼저 분석한 뒤 화면을 구현한다.
- 이름이 반복되는 Figma 레이어를 중복 컴포넌트로 단정하지 않는다.
- 새 production dependency를 설치하기 전에 사용자 승인을 받는다.
- 사용자가 요청하지 않으면 commit이나 push를 실행하지 않는다.
- 작업 완료 후 변경 파일과 검증 결과를 한국어로 보고한다.
