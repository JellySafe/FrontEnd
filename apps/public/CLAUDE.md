# Claude Code Public Rules

- 모바일 Figma 화면을 기준으로 구현한다.
- 별도의 데스크톱 정보 구조를 임의로 만들지 않는다.
- 넓은 화면에서는 모바일 웹앱 구조를 유지한 제한 너비 컨테이너를 사용한다.
- 로그인, 회원가입, 세션, 인증용 토큰 로직을 추가하지 않는다. 백엔드 연동용 익명 식별 토큰은 예외로 허용한다.
- 새 UI를 만들기 전에 `@jellysafe/design-system`에 동일한 컴포넌트가 있는지 확인한다.
- Public 전용 기능과 컴포넌트는 `apps/public` 내부에 둔다.
- 작업 완료 후 반응형 처리와 검증 결과를 한국어로 보고한다.
